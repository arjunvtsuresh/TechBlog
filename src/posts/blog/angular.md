## First Angular Project

![Angular](./images/angular/angular.jpg)

### Install Angular CLI from Command Line USing npm

```

    $ npm install -g @angular/cli

```

### Create a New App

```

    $ ng new <app-name>
    $ cd <app-name>
    $ ng serve

```

### Deploy your website on netlify

```
    $ npm install -g netlify-cli

```
